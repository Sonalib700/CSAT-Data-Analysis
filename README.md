# CSAT Data Analysis Dashboard

## Project Overview

This project analyzes Customer Satisfaction (CSAT) data to identify trends, customer feedback patterns, and key performance indicators.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Visualizations

## Key Insights

* Analyzed customer satisfaction scores across different categories.
* Identified factors contributing to higher and lower CSAT ratings.
* Created an interactive dashboard for performance monitoring.
